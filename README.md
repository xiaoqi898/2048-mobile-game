# 2048 移动版游戏

一个完全适配移动设备的 2048 游戏，支持触摸滑动和键盘控制。

## 🎮 游戏特性

- **响应式设计** - 完美适配手机、平板和桌面设备
- **触摸控制** - 支持手指滑动操作
- **键盘控制** - 支持方向键和 WASD 键
- **本地存储** - 自动保存最高分
- **流畅动画** - 平滑的方块移动和合并动画
- **精美界面** - 渐变背景和现代化 UI 设计

## 🎯 游戏规则

- 使用方向键或滑动屏幕移动方块
- 相同数字的方块碰撞时会合并
- 每次移动后会生成一个新的方块（2 或 4）
- 目标是合成 2048 方块
- 当棋盘填满且无法移动时，游戏结束

## 📱 移动端使用

- **向上滑动** - 方块向上移动
- **向下滑动** - 方块向下移动
- **向左滑动** - 方块向左移动
- **向右滑动** - 方块向右移动

## 💻 桌面端使用

- **方向键 (↑ ↓ ← →)** - 控制方块移动
- **WASD 键** - 也可以控制方块移动

## 🎨 特性说明

### 响应式设计

- 自动适配不同屏幕尺寸
- 移动端优化的小字体和间距
- 触摸友好的按钮大小

### 触摸控制

- 使用 touchstart/touchmove/touchend 事件
- 最小滑动距离阈值（50px）避免误操作
- 防止页面滚动干扰游戏

### 游戏逻辑

- 4x4 网格系统
- 智能方块合并算法
- 随机生成新方块（90% 概率为 2，10% 概率为 4）
- 游戏结束和胜利判定

## 📝 开发说明

### 修改游戏逻辑
